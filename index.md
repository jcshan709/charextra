# Charextra
Extra chars for Node.js and Typescript.  
# How to use?
Install:
```
npm i charextra
```
Pattern:
```javascript
charextra.<symbol name>
```
For example:
```javascript
const charextra = require('charextra')
const log = console.log
log(charextra.check)
```
Run it and you'll see a green tick (In fact, it's a green square root symbol)
